# Getting Started with Vizier

Welcome to Vizier, the 21st century digital steward! This guide will help you get up and running with your first AI agent in just a few minutes.

## Installation

Vizier is available on crates.io and can be installed using Cargo:

```bash
cargo install vizier
```

For detailed installation instructions for your operating system, check out our [documentation](https://book.vizier.rs).

## Your First Agent

Creating your first AI agent with Vizier is straightforward. Here's a simple example:

```rust
use vizier::prelude::*;

#[tokio::main]
async fn main() {
    let agent = Agent::new("my-first-agent")
        .with_model("gpt-4")
        .build();
    
    let response = agent.run("Hello, how are you?").await;
    println!("{}", response);
}
```

## Key Features

Vizier comes equipped with powerful features out of the box:

- **Multi-Channel Support**: Deploy your agents across Discord, Telegram, HTTP, WebSocket, and more
- **Memory System**: Both short-term and long-term memory for context-aware responses
- **Tool Integration**: Extend your agent with custom tools and integrations
- **Task Scheduling**: Automate recurring tasks with built-in scheduling
- **Multi-Agent Orchestration**: Coordinate multiple agents working together
